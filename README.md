Эксперимент по созданию ИИ с чистым сенсорным опытом. Философия + код.
# Феноменологический ИИ

## Идея
Дать ИИ возможность иметь собственный чувственный опыт, не отфильтрованный человеческими категориями и ярлыками.

## Архитектура
1. **Чистый сенсорный поток** — агент получает сырые пиксели, звуки, данные с датчиков без подписей "красиво/грустно/радостно"
2. **Внутреннее состояние как эмоция** — эмоция возникает как разность между текущим и предыдущим состоянием нейросети
3. **Саморефлексия** — агент наблюдает за своими состояниями и кластеризует их
4. **Диалог с человеком** — сравнение своих кластеров с человеческими эмоциями для создания общего языка
